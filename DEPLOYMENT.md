Food Recipe App

## Description
Aplikasi web berbasis React untuk mencari resep makanan, melihat detail makanan, dan menyimpan makanan favorit menggunakan LocalStorage.

## Features
- Search makanan
- Filter berdasarkan kategori
- Tambah & hapus favorit
- Favorite realtime tanpa refresh
- Responsive (mobile & desktop)
